    Mettre en place un partage de fichiers


    Etapes à suivre :

    1. Configurer les paramètres réseau :

      • Ouvrez le Panneau de configuration.
      • Allez dans "Réseau et Internet" > "Centre Réseau et partage" > "Modifier les paramètres de partage avancés".
      • Activez la découverte du réseau et le partage de fichiers et d’imprimantes.

    2. Partager un dossier :

      • Cliquez avec le bouton droit sur le dossier que vous souhaitez partager.
      • Sélectionnez "Propriétés".
      • Allez dans l'onglet "Partage".
      • Cliquez sur "Partager..." et choisissez les utilisateurs avec lesquels vous souhaitez partager le dossier.
      • Vous pouvez également cliquer sur "Partage avancé..." pour plus d'options.

    3. Accéder aux fichiers partagés :

      • Ouvrez l’Explorateur de fichiers.
      • Allez dans "Réseau" et vous devriez voir les ordinateurs du réseau avec les dossiers partagés.

    • Installer un système d'exploitation Windows avec un disque


    Étapes à suivre :

    1. Insérer le disque d'installation :

      • Allumez votre ordinateur et insérez le disque d'installation de Windows dans le lecteur de DVD.

    2. Redémarrer et accéder au BIOS/UEFI :

      • Redémarrez votre ordinateur.
      • Pendant le démarrage, appuyez sur la touche spécifique (souvent F2, F12, Del ou Esc) pour accéder au BIOS/UEFI de votre ordinateur.

    3. Modifier l'ordre de démarrage :

      • Dans le BIOS/UEFI, trouvez le menu "Boot" ou "Boot Order".
      • Changez l'ordre de démarrage pour que le lecteur de DVD soit en première position.

    4. Démarrer à partir du disque d'installation :

      • Enregistrez les modifications dans le BIOS/UEFI et quittez.
      • Votre ordinateur va redémarrer et devrait démarrer à partir du disque d'installation de Windows.

    5. Lancer l'installation de Windows :

      • Sélectionnez la langue, le format de l'heure et de la devise, et la méthode de clavier, puis cliquez sur "Suivant".
      • Cliquez sur "Installer maintenant".

    6. Entrer la clé de produit :

      • Entrez la clé de produit de Windows lorsque vous y êtes invité, puis cliquez sur "Suivant".

    7. Accepter les termes du contrat de licence :

      • Lisez les termes du contrat de licence, cochez la case pour accepter les termes, puis cliquez sur "Suivant".

    8. Choisir le type d'installation :

      • Sélectionnez "Personnalisée : installer uniquement Windows (avancé)" pour effectuer une nouvelle installation propre.

    9. Sélectionner la partition de destination :

      • Sélectionnez la partition où vous souhaitez installer Windows. Si vous souhaitez réinstaller sur une partition existante, vous pouvez la formater avant de continuer. Attention, cela effacera toutes les données sur cette partition.
      • Cliquez sur "Suivant" pour commencer l'installation.

    10. Installation de Windows :

      • Windows commencera à copier les fichiers et à installer le système. Cela peut prendre un certain temps.
      • Votre ordinateur redémarrera plusieurs fois au cours de ce processus.

    11. Configurer Windows :

      • Après l'installation, Windows vous guidera à travers la configuration initiale, y compris la création d'un compte utilisateur, la configuration des paramètres de réseau, et la personnalisation de certains paramètres.

    12. Finaliser l'installation :

      • Une fois la configuration initiale terminée, Windows démarrera sur le bureau.
      • Installez les pilotes nécessaires et les mises à jour de Windows pour assurer le bon fonctionnement de votre système.

    Le laboratoire Galaxy Swiss Bourdin (GSB) assure un travail de conseil et d'information auprès des prescripteurs (médecins, pharmaciens, …). Les déplacements et actions de terrain menées par les visiteurs engendrent des frais qui doivent être pris en charge par la comptabilité. Une application Web dédiée à la gestion de ces frais est en cours de développement.

    Afin d’améliorer la maintenance, la robustesse et la sécurité de ses applications Web, GSB souhaite expérimenter le développement à l’aide d’un framework.

    Dans une perspective plus large de refonte du site « GSB Frais », votre travail consiste à créer un prototype basé sur le framework Symfony. Ce prototype reprendra, dans un premier temps, les fonctionnalités qui concernent les visiteurs (consultation et saisie des fiches de frais).

Cela devrait vous permettre de démarrer plus facilement. Notez toutefois que l’accès aux données devra désormais être assuré par un ORM.

    Ce travail de prototypage doit vous amener à rédiger une note technique de type tutoriel avec la version stable actuelle de Symfony (branche 6.3). Ce document facilitera l’intégration des nouveaux développeurs dans l’équipe.

    Une fois le prototype opérationnel, vous ferez évoluer l’application vers une solution Web entièrement RWD. L’objectif étant de permettre aux visiteurs de saisir plus facilement leurs fiches de frais depuis leurs tablettes ou téléphones lorsqu’ils sont en déplacement.

    Veille technologique

    Définition

    Une veille technologique est un processus systématique et continu de collecte, d'analyse et de diffusion d'informations sur les innovations, les tendances et les évolutions technologiques. Ce processus permet à une organisation de rester informée des avancées dans son secteur d'activité ou dans des domaines connexes, afin de maintenir sa compétitivité, d'anticiper les changements et de saisir des opportunités.
